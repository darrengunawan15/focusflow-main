# 📝 ToDoList App with Firebase 

## Installation & Setup

### 1. **Clone the repository**
```bash
git clone https://github.com/darrengunawan15/focusflow.git
cd focusflow
```

---

### 2. **Install dependencies**
```bash
npm install
```

### 3. **Run the App**

```bash
npm run dev
```
The app will run at:
```
http://localhost:5173
```
